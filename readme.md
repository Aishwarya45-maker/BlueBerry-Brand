# 🫐 BlueBerry Brand

A modern and responsive fashion e-commerce website built using **HTML, CSS, and JavaScript**.

🌐 **Live Demo:**  
https://aishwarya45-maker.github.io/BlueBerry-Brand/

---

## 📖 About the Project

BlueBerry Brand is a stylish fashion e-commerce website designed to provide an elegant shopping experience. The website features a clean user interface, responsive design, smooth animations, and multiple pages for browsing products and learning about the brand.

This project was created as a frontend web development project using HTML, CSS, and JavaScript.
